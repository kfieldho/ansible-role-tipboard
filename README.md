Tipboard
=========

Install [Tipboard](https://allegro.tech/tipboard/) for a Raspberry Pi.  Tested on Rasbian Stretch.

Requirements
------------

Tested on Rasbian Stretch

Role Variables
--------------

* `tipboard_config_file` - Configuration file to tipboard to install


Dependencies
------------

None
